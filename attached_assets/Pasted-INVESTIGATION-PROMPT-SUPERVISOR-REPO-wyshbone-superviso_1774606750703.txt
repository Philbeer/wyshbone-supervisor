Search for these functions in the codebase:

Search for protocol: 'v1' in the codebase.
5. Check the executor flow after clarification
When user answers a clarification:

A new run should start (or existing run should continue)

Add logging at the start of executor execution:

6. Check database for protocol activities
Query the agent_activities table for the run:
sqlSELECT id, type, summary, created_at, 
       metadata->>'protocol' as protocol,
       metadata->>'actionTaken' as action_taken,
       metadata->>'milestoneKey' as milestone_key
FROM agent_activities 
WHERE client_request_id = '<THE_CRID_FROM_UI>'
ORDER BY created_at DESC
LIMIT 20;
If protocol column is NULL for all rows, the Supervisor isn't using the protocol helpers.
