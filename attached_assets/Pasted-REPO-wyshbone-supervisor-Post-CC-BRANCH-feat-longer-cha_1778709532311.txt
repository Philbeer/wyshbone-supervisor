CONTEXT
The chat handler currently only sees the last 8 messages in CONVERSATION
HISTORY. Older messages — including the user's original search query — fall
outside this window, so questions like "what was the first thing I asked"
can't be answered. The rolling summary helps but paraphrases, doesn't
preserve verbatim text.

Bumping the window to 50 messages costs ~2.5K extra tokens per chat call
(~$0.008 on Groq, negligible). Keeps the original query in scope for any
realistic conversation length.
