## What this guarantees

After these changes, the system NEVER goes dumb:

| Situation | What happens |
|-----------|-------------|
| User says "any" with no context | Viability guard: "I didn't catch that. What would you like to search for?" |
| Result discussion LLM fails | Graceful fallback: "I'm not sure I understood. You can ask about #3, refine, or start a new search." |
| "now try Manchester" but pattern doesn't match | Iteration fallback: "I can see you want to modify your search. You can say 'now try Brighton' or..." |
| Mission extractor produces garbage | Existing rescue LLM catches it → self-heals or asks |
| Constraint gate blocks | Smart clarification asks a good contextual question |
| Smart clarification fails | Falls back to canned constraint gate message |
| Everything fails | The Supervisor always writes SOME message to the conversation — the user never sees silence |

The principle: **every code path ends with a message to the user.** No silent failures, no garbage results, no "going dumb."
