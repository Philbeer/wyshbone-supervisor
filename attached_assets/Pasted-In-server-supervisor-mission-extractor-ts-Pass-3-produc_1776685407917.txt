In server/supervisor/mission-extractor.ts, Pass 3 produces the pass3IntentNarrative object around line 945 (the console.log that says "Pass 3 complete"). Currently this object gets passed back up the call chain and the intent_narrative artefact is only created later in mission-executor.ts around line 1036, after mission extraction, handoff diagnostics, capability checks, and mission planning have all finished.
Goal: emit the intent_narrative artefact to the user's Live Activity panel as soon as Pass 3 completes, not after the full extraction + planning chain. This lets the user see the intent card appear ~10 seconds earlier.

extractStructuredMission doesn't currently have access to runId, userId, conversationId — it only returns the trace. Add these as optional parameters so it can emit the artefact early:
tsexport async function extractStructuredMission(
  userMessage: string,
  conversationContext?: string,
  earlyEmit?: {
    runId: string;
    userId: string;
    conversationId?: string;
  },

Then, right after Pass 3 parses successfully (right after the console.log "Pass 3 complete" line, around line 945), if earlyEmit was passed, fire the artefact creation but don't await it — let it run in the background so Pass 2 can start immediately:
tsif (earlyEmit && pass3IntentNarrative) {
  // Fire and forget — don't block Pass 2 on the artefact write
  import('./artefacts').then(({ createArtefact }) => {
    createArtefact({
      runId: earlyEmit.runId,
      type: 'intent_narrative',
      title: 'Intent Narrative (Pass 3)',
      summary: `entity="${pass3IntentNarrative!.entity_description.substring(0, 80)}" scarcity=${pass3IntentNarrative!.scarcity_expectation} exclusions=${pass3IntentNarrative!.entity_exclusions.length}`,
      payload: pass3IntentNarrative as unknown as Record<string, unknown>,
      userId: earlyEmit.userId,
      conversationId: earlyEmit.conversationId,
    }).catch((e: any) => console.warn(`[MISSION_EXTRACTOR] early intent_narrative emit failed (non-fatal): ${e.message}`));
  });
  console.log(`[MISSION_EXTRACTOR] Early intent_narrative emitted to UI`);
}
